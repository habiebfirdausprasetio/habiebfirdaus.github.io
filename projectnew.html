Keunggulan Fitur

☑ Manajemen Kelas:

Fitur ini memungkinkan pengelolaan kelas secara efisien termasuk pembagian kelas, Pindah Kelas, Kenaikan Kelas

☑ Manajemen Asrama:

Mengelola asrama dengan mudah, mencakup pengaturan kamar, alokasi santri, dan wali asrama.

☑ Absensi Santri:

Sistem absensi santri yang terintegrasi untuk pencatatan kehadiran harian, baik secara manual maupun otomatis dengan Kartu Santri RFID.

☑ Manajemen Data Santri:

Fitur untuk menambah, mengedit, menghapus, dan mengelola data santri, termasuk import data dari file Excel serta pengelolaan data alumni.

☑ Manajemen Data Guru/Karyawan:

Pengelolaan data guru dan karyawan pesantren, mencakup penambahan, pengeditan, penghapusan, serta import data via Excel.

☑ Presensi Makan:

Fitur presensi makan yang memastikan setiap santri tercatat dalam daftar hadir saat makan, membantu pengelolaan dapur dan logistik serta memantau pola makan santri

☑ Presensi Sholat:

Sistem pencatatan presensi sholat yang membantu pesantren memantau kehadiran santri dalam kegiatan ibadah sholat berjamaah.
